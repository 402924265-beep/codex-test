# 制造费工作台重设计方案

2026-06-12 保存三套完整三页视觉方向，供后续改版使用。

- `option-1-executive-cockpit.png`：经营驾驶舱。适合管理层快速查看全年趋势、预算月差异和重点项目。
- `option-2-variance-matrix.png`：差异矩阵。适合财务逐月、逐小科目进行高密度分析。
- `option-3-investigation-workbench.png`：问题调查台。适合按影响排序并跟踪原因、责任人和行动闭环。

共同业务约束：

- 第一页突出全年指标趋势和主要差异来源。
- 第二页根据滚动预测版本自动关注未来三个月。例如 `4+8` 重点查看 `5月、6月、7月` 的预算与同期小科目差异。
- 第三页突出重点项目、上涨因素、下降因素和净影响。
