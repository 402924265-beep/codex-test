# V7 指标驾驶舱与三张表边界设计

## 目标

把网站稳定分成三张表对应的三部分：

1. 全年驾驶舱：看 1-12 月滚动预测、预算、同期，以及单时人效费。
2. 月度差异：看 SAP 实际与同期、预算的差异，并在小科目填写原因，上方自动总结。
3. 降费项目：只管理正式降费项目，不再承载月度差异原因。

## 指标口径

- 单：单台制造费 = 制造费 / 产量；显示 26 年、预算、同期和同比。
- 时：工作日；全年为年度工作日合计，月度为所选月工作日。
- 人：直接用人 + 间接用人的月均人数，不含白领。
- 效：UPPH = 产量 / 用人 / 工作日 / 7.5。
- 费：制造费率 = 制造费 / 产值；产值 = 平均单价 * 产量。
- 目标完成率沿用三张表：2 - 26 / 预算。
- 制造费差额 = 单台同比差 * 26 年产量。

## 数据来源

- SAP 实际报表：26 年已发生月份的小科目和大科目实际。
- 4+8 预测报表：26 年已发生月份 + 后续滚动预测。
- 姜月表：平均单价、产值、预算、同期、工作日等费用指标。
- 25 年同期基准：全年 1-12 月同期数据。

## 页面行为

### 全年驾驶舱

- 顶部单时人效费全部显示同比百分比，颜色高对比。
- 两张主图分别展示单台制造费和制造费率，均同时显示 26 年、预算、同期。
- 指标明细按单、时、人、效、费分组，支持指标、口径、月份和状态筛选。
- 鼠标悬停 26 年数据时显示同期差异、优化/恶化、预算差异和目标完成率。
- 大科目费用不再混入全年指标明细。

### 月度差异

- 月份只显示 SAP 已导入的实际月份，默认最新月份。
- 顶部显示所选月单时人效费。
- 大科目表显示同期单台、目标单台、26 实际单台、同比/目标单台差和对应影响额。
- 小科目原因在明细表填写；上方月度总结只读取这些原因自动生成。

### 降费项目

- 使用三张表“26年降费项目-洗碗机”的字段结构。
- 页面保留项目新增、删除、编辑和本地保存。
- 项目页数据不进入月度差异原因总结。

## 导出

导出工作簿继续保持三张表结构，输出全年驾驶舱、月度差异和降费项目。版式、颜色、字段顺序以现有三张表为基准。
