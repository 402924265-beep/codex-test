# 洗碗机制造费三张表工作台 - 局域网部署说明

## 一定要留好备份

请务必保留：

1. 原始压缩包 `DWWebLan.zip`
2. 正在使用的整个解压目录
3. `data/` 目录定期备份

后续如果需要修改模型、替换版本或回滚，备份文件会非常重要。`data/` 目录里保存的是同事们提交的原因和项目数据，不要随意删除或覆盖。

## 启动方式

1. 将 `DWWebLan.zip` 解压到办公电脑本地目录。
2. 双击 `Start DW Web LAN.cmd`。
3. 保持黑色命令行窗口不要关闭。
4. 本机访问：

```text
http://127.0.0.1:8780/
```

## 给同事的网址

在办公电脑上打开命令行，输入：

```cmd
ipconfig
```

找到当前公司网络网卡的 IPv4 地址，例如：

```text
192.168.1.25
```

同事访问：

```text
http://192.168.1.25:8780/
```

注意：`127.0.0.1` 只能本机访问，同事不能用。

## 共享数据逻辑

- Excel 文件仍然在每个人自己的浏览器里本地读取和计算，不上传到服务器。
- 提交原因、提交项目时，数据保存到办公电脑的 `data/` 目录。
- 一个同事提交后，其他同事刷新页面即可看到。

## 防火墙提示

第一次启动时，Windows Defender 防火墙可能提示是否允许 Node.js 访问网络。

请选择允许：

- 专用网络
- 域网络

如果不允许，同事可能无法打开网址。

## 常见问题

### 同事打不开页面

检查：

1. 办公电脑是否开机。
2. `Start DW Web LAN.cmd` 黑窗口是否还开着。
3. 办公电脑和同事是否在同一个 corp 局域网/VPN 网络。
4. IP 地址是否变了。
5. 防火墙是否拦截 8780 端口。

### 换端口

如果 8780 被占用或被拦截，可以修改 `Start DW Web LAN.cmd` 中：

```bat
set "PORT=8780"
```

例如改成：

```bat
set "PORT=8781"
```

同事网址也同步改成：

```text
http://办公电脑IP:8781/
```
