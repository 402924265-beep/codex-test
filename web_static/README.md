# 洗碗机制造费差异模型

这是纯静态网页版本。用户导入 SAP Excel 后，文件只在浏览器本地解析，不上传服务器。

## 当前版本

- 范围：洗碗机，1-4 月可选，默认 1 月。
- 口径：`26实际 - 25同期`，暂不看预算。
- 折旧：FC 已包含在“折旧（含FC）”，不会重复加。
- 4+8：只做隐藏校验，不展示明细表。

## 本地预览

在项目根目录运行：

```powershell
npm.cmd run serve:web
```

然后打开：

```text
http://127.0.0.1:8780/
```

## 发布成公网网站

把 `web_static` 文件夹作为静态网站根目录发布即可。

推荐最简单方式：

1. 打开 Netlify Drop。
2. 把 `web_static` 文件夹拖进去。
3. Netlify 会生成一个公网网址，同事都能打开。

发布后，同事打开网址并导入自己的 SAP 报表，看到的结果只在自己的浏览器里。
