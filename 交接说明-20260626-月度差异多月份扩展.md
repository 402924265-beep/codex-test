## 注意事项

1. **cooking-data.js 体积**：当前约 537KB，因为包含 5 个月的完整明细数据。如继续扩展到 12 月，文件会更大。
2. **GitHub Pages 推送**：本轮尝试推送时遇到网络问题（SSL EOF），如需公网发布，可以：
   - 重试 `git push`
   - 或直接用 `git credential-manager` 获取的 token 通过 API 上传
3. **缓存版本号**：每次修改 `cooking-data.js` 或 `app.js` 后，记得同步更新 `index.html` 里的 `?v=` 查询参数，避免浏览器缓存旧版本。
4. **Netlify 匿名部署**：如 GitHub Pages 不可用，可用 `npx netlify-cli deploy --dir=web_static --prod --allow-anonymous`，但会有密码保护（密码：`My-Drop-Site`）。

## 核心公式（业务口径）

- 单台制造费 = 制造费 / 产量
- UPPH = 产量 / (直接员工 + 间接员工) / 工作日 / 7.5
- 制造费率 = 制造费 / 产值
- 目标完成率 = 2 - 26实际 / 预算
- 制造费差额 = 单台同比差 × 26年产量
