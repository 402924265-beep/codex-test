# 厨电制造费工作台交接说明

## 一、项目目标

把现有洗碗机制造费驾驶舱扩展成多经营单元版本：

- 默认展示：厨电工厂制造费用经营驾驶舱
- 可切换：厨电工厂 / 洗碗机
- 预留经营单元：印度、巴基斯坦等海外工厂
- 当前先做公网静态网页版本，成熟后再交给 H-work 轻应用重建

项目主入口是：

- `web_static/index.html`
- `web_static/src/app.js`
- `web_static/src/styles.css`
- `web_static/src/cooking-data.js`

本地预览命令：

```powershell
npm.cmd run serve:web
```

本地预览地址：

```text
http://127.0.0.1:8780/
```

测试命令：

```powershell
npm.cmd run test:web
node --check web_static/src/app.js
node --check web_static/src/cooking-data.js
```

注意：Windows PowerShell 下不要用 `npm run ...`，可能遇到执行策略问题；请用 `npm.cmd run ...`。

## 七、关键业务公式

- 单台制造费 = 制造费 / 产量
- UPPH = 产量 / (直接员工 + 间接员工) / 工作日 / 7.5
- 制造费率 = 制造费 / 产值
- 目标完成率 = 2 - 26实际 / 目标
- 月度原因只来自小科目明细，不混入降费项目
- 降费项目单独作为正式项目库管理
